Modals (colorbox)

Table of Contents:

    Image modal

    .cb-image

    Launches a single image in an modal.

    • Add the class cb-image to your link
    • Set the href to the location of the image
    • Has maxWidth set to 80% of parent window. Height will scroll if taller than window height

    Launch image modal

    <a href="image.location" class="cb-image">View screenshot</a>

    HTML (text) modal

    .cb-html Inline html content

    Colorbox with inline HTML:
    View colorbox

    Lots of content:
    Lots of content in this colorbox

    Very little content
    Short and sweet

    Launches modal with inline html content

    • Add the class cb-html to your link
    • Set the href to "#IDofContentElement"
    • Add colorbox content to the page:
      • Add class cb-content (hides content on page load)
      • Add class cb-content-html (adds default padding)
      • Add ID with same value as href: IDofContentElement
    • Has maxWidth = 50% and maxHeight = 80%. 
    <a class="cb-html" href="#detailsModal"> View Details</a>
<div class="cb-content cb-content-html" id="detailsModal">
    This is my modal content.
</div>

    This is my colorbox element.

    It's coded on the same page as the caller link. This div has the following attributes:

    • class="cb-content" Hides content on page load, before colorbox is called.
    • class="cb-content-html" Adds default padding
    • id="IDofContentElement" This should be identical to the caller link's href.
    Hello.

    Here's some filler content

    This colorbox will fill maxheight 80% or maxWidth 50%
