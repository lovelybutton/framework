Grids

Table of Contents:

    The contains grids are a way to easily create multi-column layouts without having to use floats or write any custom css. Grids can be created with two to six column distribution and will automatically re-adjust and re-distribute their width according to their containing parent element.

    Overview

    At its core, the simplest way to create a contains grid is to code a div with the class of .contains-#. The # should be the amount of equal columns you need, from two to six. Within that div, add the correct number of div.units, one for each column:

    Coding the basic grid:

    This is a simple two-column grid:

    <div class="contains-2">
	<div class="unit"></div>
	<div class="unit"></div>
</div>

    Using other HTML elements (other than div) to build your grid:

    You can use any block-level elements for the contains grids parent or unit. Here's an example using lists as the parent:

    <ul class="contains-2 standard">
	<li class="unit"></li>
	<li class="unit"></li>
</ul>

    This list has two list items, each of which takes up 50% of its parent list. No floats!

    <div class="contains-2">
	<ul class="unit standard">
		<li></li>
		<li></li>
	</ul>
	<ul class="unit standard">
		<li></li>

    Unequal column distribution

    Unequal column distribution allows you to allocate more than one column to a unit. Let's say you need an image next to some text, but the image needs to take up 3/4 of the space. You can re-allocate the grid spacing using a size#of# class.

    Remember, your total number of .units must always equal to the number specified in the parent element. Below, we achieve four units because the first unit has a class of .size3of4. This class causes it to take up the space of three units. And the last unit takes up the last slot, leaving us with 4.

    <div class="contains-4">
	<div class="unit size3of4">This column takes up 3/4 of the available space</div>
	<div class="unit"></div>
</div>

    See example image

    Unequal column distribution cannot be used with gutters.

    Gutters

    Gutters will add a predetermined amount of space between each .unit. We have small, medium, large and extra large gutters, which are percentage-based. Simply add the class .has-gutter and the gutter sizing you want, for example medium gutter: .gutter-md to the parent container.

    <div class="contains-2 has-gutter gutter-md">
	<div class="unit"></div>
	<div class="unit"></div>
</div>

    Restrictions:

    Gutters can only be used with the basic implementation. They are not compatible with unequal column distribution

    Nested grids

    Contains grids can be nested in any configuration. The grid will automatically adapt.

    .contains-2 > .contains-4
    .unit
    .unit
    .unit
    .unit
    .unit
    <div class="contains-2 ">
	<div class="unit contains-4 ">
		<div class="unit">Lorem</div>
		<div class="unit">Lorem</div>
		<div class="unit">Lorem</div>
		<div class="unit">Lorem</div>
	</div>
	<div class="unit"></div>
</div>
