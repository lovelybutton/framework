Expandable Set

Expandable set is a UI convention that allows us to build group of expandable questions/answers. Answers are hidden on page load and are displayed when clicking on the corresponding question.

Table of Contents:

    Building the base implementation

    Expandable set relies on specific html markup, the definition list <dl>, to create the expandable segments. When a user clicks on a question, the javascript adds the class .open to both the question and answer, allowing styles to be customized based on state.

    Coding the base implementation

    1. Create the wrapper div: <div class="expandableSet"></div>
    2. Within that div, add a definition list <dl> to contain your questions and answers.
    3. (Optional) If you wish to have a controlLink (expands or collapses all questions at once), add <a class="controlThisSection">Expand All</a> anywhere in your wrapper div. Usually goes at the top.

    This is a working example of base expandable set:

    Expand All
    First section title

    The content of section 1

    Second section title

    The content of section 2

    Third section title

    The content of section 3

    <div class="expandableSet">
	<a class="controlThisSection">Expand All</a>
	<dl>
	<dt>First section title</dt>
	<dd>The content of section 1</dd>
	<dt>Second section title</dt>
	<dd>The content of section 2</dd>
	</dl>
</div>

    Named anchors (linking to a specific question)

    The script allows us to use the url hash parameter to link to and expand a specific question on page load.

    1. Add a <a name="addHashHere"></a> to the <dt>
    2. Activate the anchor by adding this to the end of the url: http://www.site.com/page/#addHashHere
    <div class="expandableSet">
	<dl>
	<dt>Question 1 <a name="addHashHere"></a> </dt>
	<dd>The content of section 1</dd>
	</dl>
</div>

    Standard style (faq)

    Coding the standard style expandable set (faq)

    1. Create the wrapper div, adding the classes expandableSet and standard:
      <div class="expandableSet standard"></div>
    2. Add the controlLink to the wrapper div: <div class="controlLinkWrap"><a class="controlThisSection">Expand All</a></div>
    3. Add your definition list <dl> containing the questions and answers.
    4. Populate each dt with:
      • The ico-swap code:
        <i class="ico swap-on-open ico-simple-blue ico-arrow-right-circled-lg ico-swap-dark-grey ico-swap-arrow-down-circled-lg"></i>
      • The FAQ question wrapped in an h4:
        <h4>FAQ question here<h4>

    This is a working example of standard faq style expandable set:

    Expand All

    Question for section 1

    The content of section 1

    Question for section 2

    The content of section 2

    Question for seciton 3

    The content of section 3

    <div class="expandableSet standard">
	<div class="controlLinkWrap"><a class="controlThisSection">Expand All</a></div>
	<dl>
	<dt> <i class="ico swap-on-open ico-simple-blue ico-arrow-right-circled-lg ico-swap-dark-grey ico-swap-arrow-down-circled-lg"></i>
	<h4>Question for section 1</h4>
	</dt>
	<dd>The content of section 1</dd>
	<dt> <i class="ico swap-on-open ico-simple-blue ico-arrow-right-circled-lg ico-swap-dark-grey ico-swap-arrow-down-circled-lg"></i>
	<h4>Question for section 2</h4>
	</dt>
	<dd>The content of section 2</dd>
	<dt> <i class="ico swap-on-open ico-simple-blue ico-arrow-right-circled-lg ico-swap-dark-grey ico-swap-arrow-down-circled-lg"></i>
	<h4>Question for section 3</h4>
	</dt>
	<dd>The content of section 2</dd>
	</dl>
</div>

    Javascript and customization

    Internationalizing the control link text

    The controlLink text is predefined in the javascript class. However, we have the ability to override the default language for a particular site, or even a particular page, simply by updating the prototype property.

    Default control link text:

    Expand All, Collapse All

    Updating the control link text:

    Add the following js to the top of your page or sitewide javascript file, outside of any document.ready function:

    fxcm.ui.ExpandableSet.prototype.controlLink.expandText = 'New expand text';
fxcm.ui.ExpandableSet.prototype.controlLink.collapseText = 'New collapse text';

    Creating a new instance

    Ocassionally, the standard expandable set functionality will not be sufficient. In these cases, instantiate a new instance of expandable set. When doing so, you can customize the wrapper ID (default is .expandableSet) and the controlLink ID (default is a.controlThisSection. You can also use the setters and getters to create more complex functionality.

    Javascript:

    $(document).ready(function(){
	var uniqueFaq = new fxcm.ui.ExpandableSet('someWrapperID', 'newControlLinkID');
	uniqueFaq.controlLink.prototype.expandText = 'Expand expand expand';
	uniqueFaq.controlLink.prototype.collapseText = 'Collapse collapse collapse';
});

    HTML:

    <div class="someWrapperID">
	<a id="newControlLinkID">Expand expand expand</a>
	<dl>
	<dt>First section title</dt>
	<dd>The content of section 1</dd>
	<dt>Second section title</dt>
	<dd>The content of section 2</dd>
	</dl>
</div>
