Backgrounds

Creating standalone emphasized content

.bg .bg-round-shadow

To create a background element a div with the class .bg is required. Within the background element a div with the class .bg-inner is used to store the content. Then an element of .bg-shoe with a child of .bg-sock is used to cap off the bottom of a background. Background elements will horizontally fill the width of their parent.

Currently there is only one style for background, described as "Rounded corners with shadow." To achieve this style a class of .bg-round-shadow is added to the background element.

.bg-align

Columns can have their bottoms justified to each other by wrapping them in a parent div with a class of .bg-align. A javascript function will calculate the largest height of all columns within that parent, and then set the height for all columns to that height.
