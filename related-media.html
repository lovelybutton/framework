Using background to construct related media

.bg .bg-round-shadow .related-media .related-media-horizontal

Creating a related media module is done by adding the relevant related media classes to a background (.bg) element.

The paperclip is a separate icon element that is inserted as a child of .bg. It has a class of .ico-bg-paperclip.
